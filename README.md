# wanAndroid
玩AndroidApp，根据鸿洋大神的玩安卓API书写，项目采用MVP+Md风格+Rxjava+Retrofit，项目实现了换肤功能，有需要的大佬可以适当的参考，本项目UI参照鸿洋API官网的Java版本的玩安卓客户端，感谢(quchao)大佬的范例
